# Oregon Overdose Deaths Dashboard

Interactive dashboard built in R Shiny to explore overdose mortality trends in Oregon from 2020–2024 using CDC SUDORS data.

## Features

- Annual overdose trends
- Drug involvement patterns
- Demographic summaries
- Circumstances surrounding overdose deaths
- Intervention indicators

## Technologies

- R
- Shiny
- bslib
- tidyverse
- patchwork

## Data Source

CDC State Unintentional Drug Overdose Reporting System (SUDORS)
